## What this project is

A web-based Reed-Kellogg sentence diagramming game. Flask backend,
PostgreSQL database, vanilla JS + SVG frontend, Docker containerized.
Current goal: refactor the codebase (module split, config externalization)
in preparation for a Slingcode port ‚Äî a self-contained single HTML file
with no server.
